📝 노트 작성 및 관리

상단 메뉴의 '홈' 탭에서 '새 노트' 버튼을 눌러 새 문서를 시작하세요. 모든 내용은 실시간으로 자동 저장됩니다.

👥 실시간 협업

'홈' 탭의 '멤버' 버튼을 통해 워크스페이스 멤버를 초대하거나, 노트 목록에서 특정 사용자를 초대하여 동시에 문서를 열고 편집할 수 있습니다.

🤖 AI 교정 기능

'AI' 탭을 선택하고 명령어를 입력한 뒤 '교정' 버튼을 누르면 AI가 문맥에 맞춰 문법 교정, 말투 변경 등을 도와줍니다.

🎨 리본 메뉴 활용

'홈', '삽입', 'AI' 탭을 전환하며 글꼴 스타일 변경, 이미지 및 도형 삽입 등 풍부한 편집 기능을 사용해 보세요.
